## 🖥️ FERRAMENTAS PARA CAPTURAR PRINTS

### Windows Built-in
- **Print Screen Key:** Copia tela para clipboard, depois colar no Paint
- **Shift + Print Screen:** Seleciona área específica
- **Snipping Tool:** Iniciar → Snipping Tool
- **Screenshot Tool:** Windows + Shift + S

### Recomendado
- **Snipping Tool:** Muito prático para selecionar apenas a janela
- **Greenshot:** Ferramenta gratuita específica para screenshots

### Passos com Snipping Tool
1. Abra Snipping Tool
2. Clique "New"
3. Selecione a área desejada (a janela do terminal)
4. Salve como PNG
5. Cole em seu documento
