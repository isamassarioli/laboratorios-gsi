💡 DICAS IMPORTANTES
═══════════════════════════════════════════════════════════════════════════════

✨ Para Capturar Prints:
   • Use Snipping Tool (Windows: Win+Shift+S)
   • Selecione apenas a janela do terminal
   • Recorte áreas desnecessárias
   • Mantenha resolução clara (não comprima)

✨ Para Criar o PDF:
   • Google Docs: Fácil, permite inserir imagens facilmente
   • Microsoft Word: Profissional, ótima formatação
   • LibreOffice: Gratuito, boa qualidade

✨ Para Melhor Visualização:
   • Use fonte clara (Arial, Calibri)
   • Tamanho de fonte: 11-12pt para texto
   • Margens: 2cm em todos os lados
   • Espaçamento: 1.5 entre linhas

✨ Para Não Errar:
   • Verifique o nome do arquivo
   • Revise a sequência de fluxos
   • Teste a abertura do PDF
   • Faça backup antes de enviar
